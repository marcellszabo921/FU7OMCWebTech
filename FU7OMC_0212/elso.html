GitHub használatának legfontosabb parancsai: Local és remote

Lokális Parancsok

1. git init: Inicializálja a Git könyvtárat a jelenlegi munkakönyvtárban.

2. git add (fájl): Hozzáadja a fájlt a következő commit-hoz.

3. git commit -m "Üzenet": Létrehoz egy új verziót a módosított fájlokból.

4. git status: Megmutatja a munkakönyvtár állapotát.

5. git log: Kilistázza a korábbi commitokat.

Remote Parancsok

1.git clone (repository_url): Letölti egy távoli repository tartalmát a helyi gépre.

2. git remote add origin (repository_url): Hozzáadja a távoli repository-t mint “origin”.

3. git push origin (branch): Feltölti a lokális változtatásokat a távoli repository-be.

4.git pull origin (branch): Letölti a távoli változtatásokat a lokális gépre.

5. git fetch: Letölti a távoli változtatásokat, de nem hajtja végre a merge-t.
